Nostr is cool!

It brings together lots of interesting technologies and use cases; this game, whilst a little ridiculous, demonstrates a few.

  1. Multi‑jurisdiction hosting: the front end of this game is served from a Sydney data centre; the back end is in a Portuguese Airbnb!
  2. Context VM: this thing is magic and, as such, we pay dev splits to the project for fun (see the donations page for payout zap receipts).
  3. A bit of applesauce: great SDK for building screens that work!
  4. A Cashu paywall: you send a token; you get a go/no‑go to move forwards in the app. Simple (although Minibits only for now; others coming soon).
  5. Splits: on the backend we account for all fees from the game and make regular payouts to the Context VM team.
  6. AI: the game engine — for what it’s worth — was created on the fly on the docks in Madeira using my Wingman agent as I laughed at a joke from the previous night.
  7. Vibe‑coding: you can just vibe‑code stuff, make donations fun, and send money to Nostr devs if you want them to feel appreciated.

Vibe‑coded with love, 2025.
